# model-evaluation

Repository for CS419 - Information Retrieval model evaluation assignment

## Prerequisite

Install Python 3.6.8 64-bit

Install package:

- stop_words
- nltk
- matplotlib
- sklearn

```sh
pip3 install nltk stop_words
```

```sh
python3
import nltk
nltk.download("stopwords")
```

## Run

```sh
python3 main.py
```
